# Branching Thunderbolt chains now show every device

When a Thunderbolt dock has two devices hanging off it, WhatCable used to follow
only one branch and silently drop the rest.

## Fixes

- **Every Thunderbolt device in the chain now shows up.** If you ran a dock with,
  say, a second dock on one port and an SSD on another, only one branch appeared
  and the other devices vanished from everything Thunderbolt-related. WhatCable now
  walks the whole fabric: the port summary lists every connected Thunderbolt device,
  the menu bar app shows the full branching tree (expandable, under "show technical
  details"), and the CLI prints a new "Thunderbolt fabric:" tree. Thanks to
  @cannotcollide for the report and the detailed data ([#280](https://github.com/darrylmorley/whatcable/issues/280)).

## Other

- Tidied up the Italian, Dutch, Turkish, and Ukrainian wording for the new
  "Thunderbolt fabric" label.
