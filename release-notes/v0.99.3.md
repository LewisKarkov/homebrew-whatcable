# Honest charging wording and community translations

## Fixes

- **The charging summary no longer reads like a live measurement.** "Charging well at 81W" showed the negotiated maximum the charger agreed to supply, not the watts actually flowing, which was confusing next to a wall meter. It now says "Charging well · up to 81W". For live measured draw, see the menu bar watts toggle or the Pro Power Monitor. Thanks @stevetrease for the careful measurements ([#317](https://github.com/darrylmorley/whatcable/issues/317)).

## Translations

- **Italian** refinements from @bovirus.
- **Traditional Chinese** improvements from @jimmyorz, including translations for strings that were still showing in English.
