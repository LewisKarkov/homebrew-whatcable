# Lighter on battery, plus a few menu bar fixes

This release cuts WhatCable's own battery use while it sits in your menu bar, and fixes a couple of things in the Power Monitor and the watts readout. Thanks to Schlomo (@k100shn) for the reports behind all of these.

## Improved

- **Much lighter on battery when idle.** WhatCable used to re-scan your ports once a second the whole time it was running, even with nothing on screen. It now backs off to once every 30 seconds when its window is closed, and returns to live updates the moment you open it. Plugging and unplugging is still picked up instantly.

## Fixed

- **Power Monitor opens on the port you're actually using.** It used to always start on the MagSafe tab, even when nothing was plugged into MagSafe. Now it opens on the connected port.
- **Pinned window stays put with the watts readout on.** If you turned on "Show charging watts" and pinned the window open, it could close on its own when the wattage number changed width. It now stays open.
- **Steadier watts readout.** The menu bar watts no longer nudge the window around as the number ticks between single and double digits.
